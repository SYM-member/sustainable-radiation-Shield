This project tested how to improve a low-cost radiation shield for measuring air temperature accurately. Five design changes were added step by step and compared to a professional Vaisala reference shield. The goal was to see how physics-based design can reduce temperature measurement error.
5
prototype design increments tested
1
reference-grade Vaisala shield used for comparison
36h
of outdoor temperature data collected for each increment
Goal
make a simple shield perform almost like a professional one
What changed in each step?

1. Black plastic pot with holes
2. Stacked plastic plates
3. White-painted outside
4. Black-painted inside
5. Double-roof design

The final design gave results very close to the professional shield.
